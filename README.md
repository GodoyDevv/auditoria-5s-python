Sistema de Auditoria 5S em Python

Projeto em Python para automatizar a análise de auditorias 5S realizadas através de planilhas exportadas do Microsoft Forms.

Objetivo

Ler uma planilha Excel contendo várias auditorias e gerar automaticamente:

contagem de respostas (Ruim, Regular, Bom, Ótimo, NA)

análise por auditoria

indicadores de desempenho

Tecnologias utilizadas

Python

Pandas

Excel
